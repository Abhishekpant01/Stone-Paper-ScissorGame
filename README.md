# Rock Paper Scissors Game

This is a simple Rock Paper Scissors game made using HTML, CSS, and JavaScript.

The player can choose Rock, Paper, or Scissors, and the computer will randomly make its choice. The game then compares both choices and displays whether the player wins, loses, or draws.

I built this project to practice JavaScript concepts such as event handling, conditionals, random number generation, and DOM manipulation.

## Features

* Play against the computer
* Instant result after every move
* Score tracking
* Simple and clean interface
* Responsive design
* YouTube tutorial/video attached on the website

## Technologies Used

* HTML
* CSS
* JavaScript
